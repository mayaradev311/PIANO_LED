# PIANO_LED


## Sobre o projeto

Este projeto foi desenvolvido na plataforma Wokwi com o objetivo de simular um pequeno piano eletrônico utilizando LEDs, botões e Arduino. O projeto permite compreender conceitos básicos de eletrônica e programação de forma prática e interativa.

Cada botão representa uma tecla do piano. Ao ser pressionado, uma nota sonora é reproduzida e um LED pode acender para indicar visualmente a tecla utilizada.

## Objetivo do projeto

O principal objetivo é ensinar como utilizar entradas e saídas digitais no Arduino, combinando sons, LEDs e botões em um único circuito. Além disso, o projeto ajuda no desenvolvimento da lógica de programação e na compreensão do funcionamento de componentes eletrônicos.

## Componentes utilizados
1 Arduino Uno
LEDs
Botões de pressão (Push Buttons)
Buzzer
Resistores
Protoboard
Fios de conexão
Funcionamento

Cada botão do circuito corresponde a uma nota musical. Quando um botão é pressionado, o Arduino identifica o comando, ativa o buzzer para reproduzir o som e pode acender um LED correspondente.

O projeto funciona como um mini teclado musical eletrônico, permitindo interação do usuário através dos botões.



## Conclusão

O projeto “Piano LED” é uma forma prática e divertida de aprender programação e eletrônica utilizando Arduino. Além de estimular a criatividade, ele ajuda a desenvolver conhecimentos importantes sobre circuitos, lógica e interação entre hardware e software.
